# Capstone frontend
Frontend for the udacity capstone project. https://github.com/Tekuor/capstone#casting-agency
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
